Caracteristicas

Los datos abstractos son estructuras de datos que se definen por su comportamiento y no por su implementación específica. Estos tipos de datos proporcionan una interfaz bien definida y ocultan los detalles de implementación subyacentes.

  • Abstracción de datos
  • Encapsulamiento
  • Ocultamiento de la implementación
  • Interfaz bien definida
  • Comportamiento específico
  • Reusabilidad
  • Flexibilidad
  • Modularidad

Listas

Una lista es una estructura de datos que almacena una colección ordenada de elementos. Puede contener elementos del mismo tipo o diferentes tipos de datos. Las listas permiten agregar, eliminar y acceder a elementos de manera eficiente. Algunas operaciones comunes en las listas incluyen agregar elementos al final, insertar elementos en una posición específica, eliminar elementos por valor o posición, y obtener el tamaño de la lista.

  • Almacenamiento secuencial
  • Acceso mediante índices
  • Capacidad de agregar y eliminar elementos
  • Flexibilidad en el tamaño
  • Posibilidad de contener elementos del mismo o diferente tipo

Tuplas

Una tupla es una estructura de datos inmutable que puede contener una secuencia de elementos de diferentes tipos. A diferencia de las listas, las tuplas no se pueden modificar una vez creadas, lo que las hace útiles para representar datos que no deben cambiar. Puedes acceder a los elementos de una tupla utilizando índices y realizar operaciones como concatenación o desempaquetado de tuplas.

  • Inmutabilidad: no se pueden modificar una vez creadas
  • Acceso mediante índices
  • Uso de paréntesis para su representación
  • Utilidad para representar datos constantes o fijos
  • Posibilidad de desempaquetar tuplas

Diccionarios

Un diccionario es una estructura de datos que almacena pares clave-valor. Cada elemento en un diccionario está asociado con una clave única que se utiliza para acceder a su valor correspondiente. Los diccionarios son útiles cuando necesitas almacenar y recuperar datos de manera eficiente utilizando una clave personalizada en lugar de índices numéricos. Puedes agregar, eliminar y modificar elementos en un diccionario, y también realizar operaciones como iteración sobre las claves o los valores.

  • Almacenamiento de datos asociados a claves únicas
  • Acceso rápido a los valores a través de las claves
  • Flexibilidad en el tipo de datos de las claves y los valores
  • Capacidad de agregar, eliminar y modificar elementos
  • Iteración sobre las claves o los valores
